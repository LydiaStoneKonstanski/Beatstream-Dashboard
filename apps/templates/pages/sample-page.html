Modeling Methods

The predictive models provide a baseline analytical counterpoint to our similarity testing model.

The first model is random and provides the control score which any intelligent model should exceed.

The second model is a naive recommendation of another song from the listenerâ€™s current artist.

The third model uses a network of artist similarities as determined by crowd-sourced surveys to recommend a song from an artist you might like.



Model Scoring

All models are scored based on the frequency with which a recommended song is interacted overall. The high score is awarded for being able to catch a favorite song as early as possible.

This same scoring was used on the similarity testing model so the effectiveness can be quantified. Each model is scored both for individual users and population as a whole leaving room for an app that could automatically select the best algorithm for each user. Every time a user chooses a song, all models provide a recommendation for the next song. Each of these recommendations is given a score for how effective it proves to be over time at predicting their later song choices. These scores can be summed at the individual user level for each model, as well as aggregated to a total score across all users. Model scores can also be tracked over time.
